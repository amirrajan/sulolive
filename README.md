## Usage
### Server
* Start server repl: ```lein repl```
* Initialize local server: ```eponai.repl=> (start-server)```

### Client
#### Web
* Start web client repl: ```rlwrap lein figwheel-web```
* Open Browser and navigate to: ```localhost:3000```
